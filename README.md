# Insurance Cost Prediction

## Project Overview

This project predicts medical insurance charges using Machine Learning.

The workflow includes data cleaning, exploratory data analysis (EDA), feature engineering, preprocessing, model training, evaluation, and residual analysis.

## Dataset

- Source: Kaggle
- File: `insurance.csv`

The dataset contains information such as:

- Age
- Sex
- BMI
- Number of Children
- Smoking Status
- Region
- Insurance Charges
